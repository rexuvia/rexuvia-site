Memory Palace Builder

Construct and navigate 3D memory palaces to store and recall information through visual-spatial association. Place memory objects, label them, and practice recall to strengthen your memory using the ancient Method of Loci.

Controls

WASD or Arrow Keys to move • Mouse to look around • Click to place memory object • Escape to exit placement mode

Click "Place Memory" to start adding objects to your palace. Each object represents something you want to remember.
